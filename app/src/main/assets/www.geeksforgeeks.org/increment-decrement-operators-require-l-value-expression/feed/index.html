Here are the operator Precedence
1) PostFix(++, –) Left to Right
2) Prefix(++,–), unary minus(-), Right to Left.

so, (i)++ will considered as (variable)increment, here we are increment a variable which has a l-value.

But, for (-i)++, first unary operator is applied to i and it would be a r-value, on which we are trying to apply postfix which complains.

Try -i++, which is equivalent to -(i++), so i is incremented and then unary is applied to result. This works fine

At same time when you try ++-i, it complains as prefix and unary have same precedence and associativity is from Right to Left, it is equivalent to ++(-i) which first evaluates -i and it would be a r-value
