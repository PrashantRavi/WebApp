But, I know a way though which we can guarantee that worst case would never occur.

We can find median of array in O(n) time using order statics algorithms. Once we have the median, we can use median as pivot. It will be an ideal pivot dividing the array in two halves. This ways the worst cases never occurs and time complexity remains O(nLogn) for all inputs. Finding median won’t increase the time complexity as finding median is O(n) which is same as partition.
