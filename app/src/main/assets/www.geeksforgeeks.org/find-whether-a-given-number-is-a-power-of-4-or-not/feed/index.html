]]> By: Anon http://www.geeksforgeeks.org/find-whether-a-given-number-is-a-power-of-4-or-not/#comment-30200 Thu, 28 Nov 2013 10:53:00 +0000 http://geeksforgeeks.org/?p=6033#comment-30200 When a number is a power of 4 , there wouldn’t be any bit in odd position, for example:
4^0 = 000 … 00001 , one at position 0
4^1 = 000 … 00100 , one at position 2
4^2 = 000 … 10000 , one at position 4
and so on.
So we just check if only one 1 bit in the number and also that bit shouldn’t be in odd position.
To check if it has only 1 bit set in the number, we can do : (n&(n-1)) == 0
To check no bit should be in odd position : (n&0xAAAAAAAA)==0

So solution is :

bool check(unsigned int n){
return (n&(n-1)) == 0 && (n&0xAAAAAAAA) == 0;
}
