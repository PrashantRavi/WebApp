Anyways, printf() will be excuted in right to left fashion. so first it comes across ‘ i ‘, so value of ‘ i ‘ is 2. Next it comes across i–, so value printed in this position will be 2 but the value of i is 1, Next –i, so value here will be 0 and value of i is also zero.

so o/p should be 0,2,2 but the value of i is currently 0, as the value of i is changed in the memory the o/p will be 0,2,0

Always remember ‘i’ is a variable and ‘i++’ (and like) are literals.

Now, u can try out this statements too..

1)printf(“%d %d %d n”, –i,i–,i++);

2)printf(“%d %d %d n”, –i,i–,++i);

Here we are not printing variables (not from memory), just printing the literals after applying operators.
