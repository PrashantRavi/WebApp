And from all such sub arrays we are choosing the min length sub array using min_len. Here is the example,
Suppose {1,4,45,6,52,10,19} and k=51.
start=0,end=0. After first inner while loop , sum = 1+4+45+6 = 56>51 and end=4. After second inner while loop, sum = 45+6 start = 2 end=4.
min_len = 3.
But this is not the required sub array as it is not of minimum length. If you remove the outer while loop, program will return this as min sub array.Due to outer while loop, it continues to check for other sub arrays (end<n). And finally by using min_len we can track the min_len sub array and return that as the required sub array. Here the answer should be {51} not {4,45,6}.
