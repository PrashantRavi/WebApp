In this case, we start from the end of the array .As we know, NGE of the righmost element will be always -1. We push the righmost element into the stack (as it is a candidate for being NGE of some other element present in its left).After this, we process elements one by one from size-2 downto 0.We compare the current element with the top of the stack .if the top of the stack is greater than the current element ,then top is the NGE of the current element and we push the current element into the stack,if the top of the stack is less than the current element then we repeatedly pop the elements from the stack until top of the stack becomes greater than the current element or stack becomes empty.if stack becomes empty, then NGE of current element is -1 otherwise NGE of current element is top of the stack.then we push the current element into the stack.Thus, each element is pushed and popped atmost once even in the worst case (10,9,8,7) or (1,2,3,4).
