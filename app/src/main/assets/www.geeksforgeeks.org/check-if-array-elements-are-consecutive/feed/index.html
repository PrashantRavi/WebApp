It follows the following 5 steps
1.calculate the sum of given array
2.find min element in array and find the sum below the minimum element ( till min-1) using formula n*(n+1)/2
3.assing max=min+n (supposed to be maximum if array elements r consecutive). calculate sum of elements from 1 to max using max*(max+1)/2
4. find the difference between sum till maximum and sum below minimum.
5. if result produced by step 4 and step 1 are equal then the array elements are consecutive or else not
