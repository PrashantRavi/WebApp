]]> By: GeeksforGeeks http://www.geeksforgeeks.org/total-number-of-possible-binary-search-trees-with-n-keys/#comment-20285 Fri, 21 Jun 2013 19:31:52 +0000 http://geeksforgeeks.org/?p=7695#comment-20285 @All:

The above fact is correct. The number of possible Binary Search Trees with n different keys is Catalan number only.

Number of structurally different binary trees is also a catalan numbers.

They both are different types of questions, but their result is same.

In first, we are given n different keys and we want to know the number of ways we can organize them to form a BST. Let us take few examples.
keys[] = {1, 2, 3}, we can organize them in 5 ways: two trees with 1 as root + 1 tree with 2 as root + 2 trees with 3 as root.
Take more examples, and you will see pattern of catalan numbers. See question 2 of this for more details.

In second, we want to know structurally different trees with n nodes, we are not given keys.
For n = 3, there can be 5 trees. The expression for this will be same as the expression for BST provided here
