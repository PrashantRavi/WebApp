1. Maintain a queue of all nodes that have been seen so far.
2. Starting with root, add a node to the queue in the order that it was visited.
3. After an iteration of insertion, pick the queue-head and print. Discard queue-heads until a head with CHILDREN is found. Add these Children to the queue and continue until a counted integer reaches the height of the tree.

For a right-view, i would assume changing the DS to a stack should suffice.
