First we will search for index of 17 –> 4 let say it is x. so x=4.
Initially,
index_diff = k
left = right = x

Now, (in the loop)
index_diff = index_diff/2 (2)
left = left-index_diff (2)
right = right+index_diff (6)

We will compare the values at x+index_diff and x-index_diff w.r.t. diff
i.e. A[right]-A[x] and A[x]-A[left] ==> 13 > 3
so we added k/2 elements (14,15)

Next iteration,
index_diff = index_diff/2 (1)
left = left-index_diff (1)
right = right-index_diff (5)

Now we will compare again A[right]-A[x] and A[x]-A[left] ==> 7 > 4
so we have added k/4 elements. (13)

and continue until we include k elements.
This will take O(log k) as we need only the indexes.
