// max difference of two elements in the array
// s.t. larger element appears after the smallest
// element in the array
int maxDiff(int arr[],int size) {
int i;
int max_diff = arr[1]-arr[0];
int min_element = arr[0];
for (i=1;i min_element) {
// diff of current and min element
int diff = arr[i]-min_element;
if (diff > max_diff) {
// update max diff
max_diff = diff;
}
}
else {
// update the min element in the array
min_element = arr[i];
}
}
return max_diff;
}

It can be done this way also.
2, 3, 10, 6, 4, 8, 1
Traverse from last element. For every element keep track of biggest element (say max) in its right side and also difference of that element with max.

So in this example,
for 1, no max and hence no max difference
for 8, 1 is max and diff is 1-8= -7, so max diff is -7
for 4, 8 > current max(1), so max=8 and diff=8-4=4 and 4 > current max diff(-7), so current max diff = 4
for 6, 4 < curr max(8), so max is still 8 and diff = 8-6=2 and 2 < curr max diff(4), so curr max diff is still 4
for 10, 6 < curr max(8), so max is still 8 and diff = 8-10=-2 and -2 curr max (8), so now max = 10, and diff=10-3=7 and 7>curr max diff(4), so update max diff to 7
for 2, curr max= 10, and diff = 10-2=8 and 8>curr max diff(7) , so max diff = 8
which is the final answer.

O(n) time and constant space.
