]]> By: anonymous http://www.geeksforgeeks.org/write-a-program-to-reverse-an-array-or-string/#comment-30588 Mon, 09 Dec 2013 12:20:00 +0000 http://geeksforgeeks.org/?p=2567#comment-30588 why so much trouble? You are passing two additional variables everytime when you recurse. In a way, using extra space.
There is a pretty standard to swap without using the third (temp) variable,

1.) a=a*b;
2.) b=a/b;
3.) a=a/b;

Note: This will work for both positive and negative integers. But for large integers, may cause an overflow, in which case, either switch to long long or use + instead of * and – instead of /.
