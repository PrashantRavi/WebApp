The code for the above problem is:
void equilibrium(int a[],int n)

{
int left[n],right[n],temp;
left[0]=0;
right[n-1]=0;
for(int i=1;i=0;j–)
{
temp=a[j+1];
right[j]=right[j+1]+temp;
}
for(int i=0;i<n;i++)
{
if(left[i]==right[i])
printf("Equi %d",i);
}
}

Time complexity: O(n)
Space complexity: O(n)
Not much efficient considering the space complexity but is another approach which i think can be included in the post.
