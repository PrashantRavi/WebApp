]]> By: AlienOnEarth http://www.geeksforgeeks.org/reservoir-sampling/#comment-37977 Mon, 09 Jun 2014 08:06:00 +0000 http://www.cdn.geeksforgeeks.org/?p=25866#comment-37977 This algorithm takes constant space and linear time.

Why does it work? Let’s say you have three elements.
The probability of choosing the first element (by the end) is P(not replacing with second element) * P(not replacing with third element) = 1/2 * 2/3 = 1/3
The probability of choosing the second element is P(replacing first element with second element) * P(not replacing second element with third element) = 1/2 * 2/3 = 1/3
The probability of choosing the third element is P(replacing an element with the third element) = 1/3

In general you can prove correctness by induction. Suppose that after seeing n-1 elements, each of the first n-1 elements is chosen with equal probability. When you see the nth element, it is chosen with probability 1/n, and each of the other elements is chosen with probability [1/(n-1)] * [(n-1) / n] = 1/n. Therefore all elements are chosen with equal probability.
