int isPalindrome(unsigned int x) {
int i;
for(i = 31; i >= 0; i–) {
unsigned int mask1 = 1 << i;
unsigned int mask2 = 1 << (31 – i);
if((mask1 & x) != (mask2 & x))
return 0;
}
return 1;
}

In the given approach, x is being left shifted whereas in my approach I am taking two mask(s), one pointing to the left end of the string and other pointing to the right end of the string and comparing if the set bits are same or not.
