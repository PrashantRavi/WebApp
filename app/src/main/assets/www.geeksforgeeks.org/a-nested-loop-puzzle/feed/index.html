If we apply the concepts of locality of reference, the performace evaluation will be different. Consider we are accessing a static variable inside the body of inner loop. Think of it as huge buffer containing an image data.

There are two types of locality of reference, temporal and spacial. Since the loop code is condensed, it will have good space and temporal locality. However, there will be penality every time when the static variable is accessed, since it could be sotred in data section (most probably different page).
