]]> By: shelly http://www.geeksforgeeks.org/segregate-even-and-odd-numbers/#comment-27083 Wed, 28 Aug 2013 14:22:00 +0000 http://geeksforgeeks.org/?p=7897#comment-27083 void segregateEvenAndOdd( int A[], int N ) {
int countOfEven = -1, i = 0;
while ( i < N ) {
if ( !( A[i] & 1 ) )
swap ( A[i], A[++countOfEven] );
i++;
}
}
PS: ordering is also maintained.
