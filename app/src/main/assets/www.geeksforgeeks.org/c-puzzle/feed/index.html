]]> By: Viji http://www.geeksforgeeks.org/c-puzzle/#comment-28127 Thu, 26 Sep 2013 06:55:00 +0000 http://www.cdn.geeksforgeeks.org/?p=123206#comment-28127 For this, you need change the return address. The following code will work with gcc compiler.

For 64 bit machine,
void fun()
{
int j = 10;
int *p = NULL;

// take the rbp register in p
asm(“movq %%rbp, %0”: “=r” (p):);

// Move the pointer to the next position in the stack
// where the return address would get stored
p += (sizeof(p) / sizeof(*p));

// Since, printf is the next instruction of ‘i = 20’
// increment the return address by 4 bytes.
*p = *p + 4;
}

For 32 bit machine, you can use ebp instead of rbp in the inline assembly code.
