]]> By: Venki http://www.geeksforgeeks.org/c-internals-default-constructors-set-1/#comment-5392 Mon, 19 Sep 2011 14:23:03 +0000 http://geeksforgeeks.org/?p=10857#comment-5392 Constructor meant for setting an initial state of object. In case of inheritance hierarchy, base class objects will be constructed in the order of inheritance hierarchy (IS-A relation in OO terminology) followed by derived class objects.

Similarly when an object is embedded in another object (HAS-A relation in OO terms or containment), constructors of embedded object are called in the order of their declaration.

In the above case, A is embedded in D. A’s constructor to be called to setup state of ‘a’, prior to constructing ‘d’ object.
