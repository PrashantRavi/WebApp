]]> By: kartik http://www.geeksforgeeks.org/functions-that-are-executed-before-and-after-main-in-c/#comment-5559 Sat, 01 Oct 2011 04:32:22 +0000 http://www.cdn.geeksforgeeks.org/?p=14538#comment-5559 This constructor is not a c++ type of constructor. In C++, constructors are used to initialize objects of a class.

Here constructor is an attribute added to functions so that they are executed before main(). In fact, there can be more than one functions with constructor attribute.

 #include<stdio.h>
void myStartupFun (void) __attribute__ ((constructor));
void myCleanupFun (void) __attribute__ ((destructor));

void myStartupFun2 (void) __attribute__ ((constructor));
void myStartupFun2 (void) __attribute__ ((destructor));

void myStartupFun (void)
{
    printf ("startup fun() before main()\n");
}

void myCleanupFun (void)
{
    printf ("cleanup fun() after main()\n");
}


void myStartupFun2 (void)
{
    printf ("startup fun2() before main()\n");
}

void myCleanupFun2 (void)
{
    printf ("cleanup fun2() after main()\n");
}

int main (void)
{
    printf ("hello\n");
    return 0;
}

Output

startup fun2() before main()
startup fun() before main()
hello
cleanup fun() after main()
startup fun2() before main()
