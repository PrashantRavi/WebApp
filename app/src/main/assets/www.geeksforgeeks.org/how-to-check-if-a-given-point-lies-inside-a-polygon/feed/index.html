considering vertices on the ray as lying slightly above the ray.

This is how you fix it :-
replace the following piece of code in the section tagged “general case” :-


if (o1 != o2 && o3 != o4)
return true;

with :- (This code now detects intersection ONLY if the second second vertex of edge is below the extreme ray)

if (o1 != o2 && o3 != o4)
{
if (!o4)
{
return false;
}
if (!o3)
{
if (o4 != 1)
{
return false;
}
}
return true;
}
