Create a HashMap of the type <String, List>

The key string here will be the sorted string and list will contain all the strings from the input array which corresponds to the sorted key. For example:
cat and tac will have the same key as act. So its hashmap entry will be
act : cat -> tac

At the end, just traverse over the entire keyset of the hashmap.
