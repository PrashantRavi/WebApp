Use cumulative sum. Modify original array such as arr[i]=arr[i]+arr[i-1].
For ex.—> arr–is– 4 2 -3 1 6
After cumulative sum it would be —- 4 6 3 4 10
For given sum as “sum” (here sum is 0), For every arr[i] from back search for arr[i]-sum in array. If found at index j then j+1 to i is required subarray with given sum.
In ex. given we found 10. So search for (10-0=10) not found. Go for i=3 i.e.arr[i]=4. 4 is found at index 0.
So (0+1,3) is required subarray with sum zero.
NOTE:
1.this method works for any given sum and both positive and negative numbers.
2. Use some smart search or use hashing to search arr[i]-sum!
