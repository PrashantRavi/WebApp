STEPS FOR DETERMINING SINGLY CONNECTIVITY
->run dfs from vertex i
->classify tree,forward,and cross edges in that dfs
->eneque the vertex in a queue
while queue is not empty
v=q.del()
find all the ancestors of the verices including itself
if there exist a forward edge or cross edge on any one of its ancestor then there is more than 1 path….return false
if not eneque all its child
repeat the process while queue is not empty
->do it for all d vertices
