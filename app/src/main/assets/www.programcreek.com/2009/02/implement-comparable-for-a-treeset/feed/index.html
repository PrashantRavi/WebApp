class Dog{
// other sections of the code ommitted
public static final Comparator dogSize = new Comparator(){
@Override
public int compareTo(Dog o) {
return size - o.size;
}
}
}

And in the main method, you can use it as


TreeSet d = new TreeSet(Dog.dogSize);
