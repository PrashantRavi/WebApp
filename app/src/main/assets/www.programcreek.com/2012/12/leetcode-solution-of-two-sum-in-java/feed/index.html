]]> By: Tom Dawn http://www.programcreek.com/2012/12/leetcode-solution-of-two-sum-in-java/#comment-8877 Wed, 03 Feb 2016 03:45:00 +0000 http://www.programcreek.com/?p=10560#comment-8877 The solution here is a bit tricky. Consider an array [4,1,6] where target equals 10. What it does is put complementary number of numbers[i] into the hashmap together with index value i. Here, 10-4=6, so it is key 6 value 0. That 6 is really what we are looking for to add up to 10, since we already have a 4. Later on if it finds 6, it will simply return the index of the previous complementary number and index of 6, which is 0+1 and 2+1.
