Face Collage Maker 2.0 Download

 


What is Face Collage Maker

Face Collage(or collage face) Maker is a tool for creating a collage face by using many small pictures. Photoshop is a good tool for photo processing, but it is not good for making face collage. The Face Collage Maker was developed several years ago, and it has been totally free. It has been downloaded thousands of times during last several years. The current version is much faster than the original one.

Face Collage Maker is a tool for converting the left face picture to the right face collage picture which consists more than 1000 small pictures.

 
Besides the demo above, you can use face collage maker by using different repositories, e.g. project a group of animals on a face, a bunch of faces on a face, etc.

The face below is composed by using a lot of dog pictures, when zoom in, you can see a lot of cute dogs.

Download

On most windows operating systems, you should be able to launch it by double-clicking run.exe which is under FaceCollageMaker directory. It is on a free file server website - hotfile.com. The download link will lead you directly to a zip file.



Download Link 1 (Full Version with picture repository)
Download Link 2 (Short version without picture repository)

How to use it?

1. Select a face picture as your object to project small pictures.
2. Select a picture repository where you will need a bunch of pictures.
3. Select a destination directory location where you want to store the generated collage picture.
4. Click the "Generate" Button, then wait and get your collage picture!

Video Demo in 2 minutes

Tips for Better Results

1. The more pictures you use as source pictures, the better the quality you will get for your faces.
2. Cast only the face part of the main picture.  This enables you have a focus on a face only, and therefore has better quality.
3. Use various kinds of pictures. This means better options for projection on each part of face picture.

Possible Problems

1. If pictures in your repository are large (more than 3MB), the collage maker will take more time to generate collage.
2. System currently only support .jpg files.
3. Total time required also depends on the size of your repository.
