Test Scenarios

A test scenario is a high level documentation of a use case. A single test scenario can cater multiple test cases. We use scenario testing technique to test the different use cases of the application, covering its business requirements.
Below is the list of test scenario examples frequently asked in interviews. Although there could be numerous scenarios to be tested in the system/applications referred below but we have limited the scenarios to the most basic and generic functionalities.
